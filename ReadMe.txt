This project is a demo to utilize the main 6+1 SASS features: 
1) Creation and Using variables. 
2) Nesting.
3) Inheritance (using @extend). 
4) Splitting code into multiple files (using @import). 
+1) Various functions such as color functions (darken(), lighten()). 
5) Mixins (@mixins and @include).
6) Operators.
